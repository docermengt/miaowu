# movie

 介绍
             仿猫眼电影javaweb项目

#### 软件架构

本项目采用B/S架构，由tomcat容器运行，前后端分离

以下为项目技术栈： 由maven3.6管理第三方jar包

​     1.Java1.8         

​      2.mybatis3.5                               

​     3.javax-servlet-api

​     4.javax-jsp-api

​     5.fastjson-1.26

​     6.jsp页面需引入jQuery包


#### 运行教程
```
1.在pom文件中导入相关依赖
2.idea中添加tomcat本地服务器部署工件
3.端口号8081 项目路径：movie
```
